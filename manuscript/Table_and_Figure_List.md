# Tables and Figures

## Main Tables

**Table 1. Summary of validated adenovirus genomes recovered from animal-associated metagenomic samples.**
Includes host, sample ID, virus assignment, closest reference, genome coverage, variant counts, and phylogenetic placement.

**Table 2. Evidence layers supporting adenovirus genome recovery.**
Summarizes read mapping, coverage breadth/depth, reference-guided consensus reconstruction, BLAST confirmation, phylogenetic placement, variant annotation, and independent de novo assembly support.

---

## Figures

**Figure 1. Genome-wide coverage profiles of validated adenovirus genomes.**
Coverage profiles for the goat-derived AdV-12-like genome and dog-derived AdV-18-like genome showing read depth across each reference genome.

**Figure 2. Rooted expanded adenovirus phylogeny.**
Maximum-likelihood phylogeny including study genomes, closest GenBank relatives, African contextual sequences, Nigerian clinical AdV-40 sequences, additional Human mastadenovirus A references, and canine/bovine outgroups.

**Figure 3. Variant burden by gene/product.**
Bar plots showing the genes/proteins with the highest number of coding variants in each recovered genome.

**Figure 4. SNV consequence profiles.**
Summary of synonymous, missense, mixed-overlap, noncoding, and other SNV consequence categories in each validated genome.

**Figure 5. Missense versus synonymous coding SNVs by gene/product.**
Stacked bar plots showing the distribution of synonymous and missense substitutions across the most variable genes.

**Figure 6. Genome-wide variant landscape.**
Genome-coordinate plots showing the distribution of high-confidence synonymous and missense SNVs across each recovered adenovirus genome, with coding regions shown as genome features.
